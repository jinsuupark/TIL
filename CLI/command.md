# COMMAND

command line interface 명령어를 정리합니다.



#### pwd

현재 위치한 폴더를 출력하는 명령어입니다.

``` shell
pwd
```



#### ls

list의 약자로 현재폴더에있는 파일과 폴더를 출력하는 명령어입니다.

```shell
ls # 숨김폴더는 출력안함.
ls -a #숨김폴더,파일까지 출력함.
```



#### cd

change directory의 약자로 폴더를 이동하는 명령어 입니다.

```shell
cd <PATH> #이동하고싶은 폴더를 path에 입력
```



생성

- `touch` : 파일을 생성
- `mkdir`: 폴더를 생성
