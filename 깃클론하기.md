``` 
git clone <git 주소>
```

(git 폴더가 같이옴)



```
git remote remove origin
```

개발자가 저장한 루트를 삭제

```
git remote add origin https://github.com/jinsuupark/jinsuuupark.github.io.git
```

나의것을 추가

```
 git push origin master
```
